# Steam Tools

### Setup guide:
```
git clone
python3 -m venv env
pip install requirements.txt
```
Create 2 files:
* steam_accounts.txt
  - File with login and password in format:
    ```
    login1 password1
    login2 password2
    ...
    loginN passwordN
    ```
* steamids.txt
  - File with steam ids that will be added in format:
    ```
    steamid1
    steamid2
    ...
    steamidN
    ```
